## DESIGN PHILOSOPHY

### Core Principle: Every Page Is a Mini Comic
Each of the 22 story pages works like a **4–6 panel comic strip**. No full-page illustrations. Instead, small sequential panels tell the story like a cartoon strip — exactly how kids understand stories best.

### Target Audience
- **Primary (Kids 2–5):** Cannot read. Panels guide the eye left → right, top → bottom. Big expressive faces, clear actions, bright colors.
- **Secondary (Parents):** Want a clear moral they can discuss. The **Modern Life Moral** on each page makes it easy for parents to connect the story to today's life.

### Design Rules

| Rule | Implementation |
|---|---|
| **80% Illustration / 20% Text** | Each panel has max 1 short caption (5–8 words). Title + Moral box below panels. |
| **Comic Panel Grid** | 4 panels = 2×2 grid. 5 panels = 2+3 or 3+2. 6 panels = 2×3 or 3×2 grid. |
| **Big Faces, Big Emotions** | All characters drawn in **chibi/kawaii** style — oversized heads, huge eyes, exaggerated expressions |
| **No Scary Imagery** | Ravana = grumpy/silly villain. Battles = cartoon action. No blood, no fear. |
| **Warm Indian Palette** | Saffron, golden yellow, sky blue, forest green, lotus pink — zero black or grey |
| **Consistent Characters** | Same chibi style for all characters across all 22 pages |
| **Modern Life Moral** | Every story ends with how the lesson connects to a child's **real daily life today** |
| **Speech Bubbles** | Key character lines are shown in round speech bubbles inside panels |
| **Panel Numbering** | Subtle ① ② ③ numbers in corner of each panel to guide reading order |

## PANEL ILLUSTRATION PRODUCTION NOTES

### For the Illustrator

1. **Consistency First:** Create a **master character sheet** before drawing a single panel. All 7 core characters must look identical across all 22 pages.
2. **Panel Emotions:** Each panel tells ONE emotion clearly. Don't crowd panels with multiple emotions.
3. **Reading Flow:** Panels must be numbered ① ② ③ visually and designed for left to right, top to bottom reading.
4. **Speech Bubbles:** Round bubbles for speech. Cloud-shape for thoughts. Keep max 6 words per bubble.
5. **Action Lines:** Use comic-style speed lines behind characters for dynamic action moments.
6. **Expressions Library:** For each character, create an expression reference sheet — Happy, Sad, Angry, Scared, Brave, Proud, Surprised. Use consistently.
7. **Background Complexity:** Simple in panels — one clear background element (forest, palace, river). No busy backgrounds that compete with characters.
8. **Moral Box Design:** Every page — same lotus-border box at bottom. Saffron background, white bold text. Like a stamp the child can always find.

### Software and Technical Specs
- **Format:** Adobe Illustrator (vector) preferred for print quality
- **Resolution:** 300 DPI minimum, CMYK color mode
- **Bleed:** 3mm all sides | Safe Zone: 5mm inside trim
- **Trim size:** 210mm × 148mm (A5 Landscape)
- **Panel borders:** 2pt rounded-corner saffron border
- **Inter-panel gutters:** 4mm
- **Rich black:** C:40 M:30 Y:30 K:100 for deep blacks
