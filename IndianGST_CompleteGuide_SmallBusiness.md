## 1. GST FUNDAMENTALS

### What is GST?
GST (Goods and Services Tax) is a **single, unified indirect tax** that replaced VAT, Service Tax, Excise Duty, and 13+ other taxes from July 2017. It follows a **destination-based consumption tax** model.

### The Three Types of GST
| Tax | Full Form | When Applied |
|-----|-----------|--------------|
| **CGST** | Central GST | Sales within same state (50% goes to Centre) |
| **SGST** | State GST | Sales within same state (50% goes to State) |
| **IGST** | Integrated GST | Sales across states OR exports |

**Example:**
- You sell in Karnataka to Karnataka customer → CGST (9%) + SGST (9%) = 18%
- You sell in Karnataka to buyer in Maharashtra → IGST (18%)
- Amazon India: You're selling via marketplace, IGST typically applies

### GST Slabs
- **0% / Exempt** — Books, fresh food, medicines (essential goods)
- **5%** — Footwear below ₹1000, packaged food
- **12%** — Stationery, notebooks, processed food
- **18%** — Electronics, most manufactured goods, services
- **28%** — Luxury goods, automobiles, tobacco

### Key Terminology
| Term | Meaning |
|------|---------|
| **GSTIN** | Your 15-digit GST Identification Number |
| **Outward Supply** | Your sales (you charge GST) |
| **Inward Supply** | Your purchases (vendor charges you GST) |
| **ITC** | Input Tax Credit — GST paid on purchases, offset against GST collected on sales |
| **Nil-Rated** | 0% GST, ITC cannot be claimed |
| **Exempt** | Outside GST scope, ITC cannot be claimed |
| **Zero-Rated** | Exports — 0% but ITC CAN be claimed (refund mechanism) |
| **Composition Scheme** | Simplified scheme for small businesses |
| **QRMP** | Quarterly Return Monthly Payment scheme |
| **RCM** | Reverse Charge Mechanism — you pay GST instead of vendor |

## 2. GST REGISTRATION

### Mandatory Registration Thresholds
| Business Type | Annual Turnover Threshold |
|---------------|--------------------------|
| Normal goods supplier | ₹40 lakhs (₹20L for special states) |
| Services provider | ₹20 lakhs |
| **E-commerce seller (Amazon/Flipkart)** | **MANDATORY regardless of turnover** |
| Inter-state supplier | MANDATORY regardless of turnover |

> **Your Case:** Since you sell on Amazon India, **registration is mandatory** even if your turnover is ₹1.

### How to Register
1. Go to **[gstin.gov.in](https://www.gstin.gov.in)** → New Registration
2. Part A: Mobile + Email OTP verification → Get TRN (Temporary Reference Number)
3. Part B: Fill business details within 15 days using TRN
4. Documents needed:
   - PAN card
   - Aadhaar card
   - Bank account details + cancelled cheque
   - Business address proof (electricity bill / rent agreement)
   - Photograph
5. ARN (Application Reference Number) issued
6. GSTIN issued in 7-10 working days

### Your GSTIN Structure (15 digits)
```
29  AAAAA1234A  1  Z  5
↑   ↑           ↑  ↑  ↑
State PAN      Entity Check
Code           No. digit digit
```
- 29 = Karnataka
- First 2 digits = state code

## 4. GST RETURNS FILING

### Types of Returns You Need to File

| Return | What It Reports | Due Date | Who Files |
|--------|----------------|----------|-----------|
| **GSTR-1** | Your outward sales | 11th of next month (Monthly) / 13th (Quarterly) | All regular taxpayers |
| **GSTR-3B** | Summary + tax payment | 20th of next month | All regular taxpayers |
| **GSTR-2B** | Auto ITC statement | Auto-generated on 14th | Read only |
| **GSTR-9** | Annual return | 31st December (following FY) | All with turnover >₹2Cr |
| **GSTR-9C** | Reconciliation + Audit | 31st December | Turnover >₹5Cr |

### Monthly vs Quarterly — Which One for You?

#### QRMP Scheme (Quarterly Return Monthly Payment)
- **Eligible if:** Annual turnover ≤ ₹5 crores in previous FY
- **What's quarterly:** GSTR-1 and GSTR-3B
- **What's monthly:** Tax payment (IFF — Invoice Furnishing Facility for GSTR-1)
- **Benefit:** Less compliance, file returns only 4 times/year instead of 12

#### Regular Monthly Scheme
- File GSTR-1 and GSTR-3B every month
- Better if your business has complex transactions or high turnover

> **Recommendation for SanskaTots:** Start with **QRMP scheme** since you're likely under ₹5Cr turnover. Pay tax monthly but file quarterly.

### Payment Due Dates (QRMP)
| Month | Payment Due |
|-------|------------|
| Q1 (Apr-Jun) | Pay by 25th May, 25th June; File by 22nd/24th July |
| Q2 (Jul-Sep) | Pay by 25th Aug, 25th Sep; File by 22nd/24th Oct |
| Q3 (Oct-Dec) | Pay by 25th Nov, 25th Dec; File by 22nd/24th Jan |
| Q4 (Jan-Mar) | Pay by 25th Feb, 25th Mar; File by 22nd/24th April |

**Note:** 22nd for Category 1 states (Karnataka = 24th)

## 5. SELF-FILING GUIDE (Step by Step)

### Tools You Need
1. **GST Portal:** [gst.gov.in](https://www.gst.gov.in) — Official portal, free
2. **GSTN DSC / EVC:** Electronic Verification Code (use Aadhaar OTP for small businesses — easiest)
3. **Tally / Zoho Books / Excel** — For maintaining records
4. **ClearTax / TaxBuddy** — Simplified filing interfaces (free tier available)

---

### STEP 1: Prepare Your Sales Data (Before Filing GSTR-1)

Create a spreadsheet with:
```
Date | Invoice No | Customer Name | Customer GSTIN | State | HSN Code | 
Taxable Value | CGST | SGST | IGST | Total | Intra/Inter State
```

**For your business (books):**
- All book sales = Exempt Supply = ₹0 tax
- You still need to report these in GSTR-1 under "Exempt/Nil" column
- B2B sales (if selling to registered businesses) → listed separately
- B2C sales (direct customers) → listed as B2C large (>₹2.5L) or B2C aggregate

---

### STEP 2: File GSTR-1

1. Login → gst.gov.in → Returns Dashboard
2. Select Financial Year + Tax Period
3. Choose GSTR-1
4. Fill sections:
   - **4A, 4B, 4C** — B2B invoices (sales to GST-registered buyers)
   - **5A, 5B** — B2C large invoices (>₹2.5L, inter-state)
   - **7** — B2C small (aggregated)
   - **8** — Nil/Exempt/Non-GST supplies ← **Your main section**
   - **12** — HSN-wise summary
5. Preview → Submit → File using EVC (Aadhaar OTP)

---

### STEP 3: File GSTR-3B

This is the **most important return** — payment happens here.

1. Login → Returns Dashboard → GSTR-3B
2. Section 3.1 — Tax on outward supplies:
   - 3.1(a): Outward taxable supplies (rate-wise breakup)
   - **3.1(b): Outward taxable (zero rated)** — Exports
   - **3.1(c): Other outward supplies (Nil/Exempt)** ← Your books go here
   - 3.1(d): Inward supplies (RCM)
   - 3.1(e): Non-GST outward supplies
3. Section 4 — ITC available:
   - For your business → likely nil or reversed (see Section 6)
4. Section 5 — Values of exempt/nil/non-GST:
   - Report your book sales value here
5. Section 6 — Payment of tax
6. Preview → Pay → File

---

### STEP 4: Payment

- Go to **Electronic Cash Ledger** on GST portal
- Create Challan → Select CGST/SGST/IGST amounts
- Pay via Net Banking / UPI / NEFT
- Credit reflects in 1-2 hours (NEFT same day)

---

### STEP 5: Reconcile Monthly

After filing, reconcile:
- Your books of accounts vs GSTR-1 filed
- GSTR-2B (auto) vs your purchase invoices
- Cash Ledger balance

## 6. YOUR SPECIFIC CASE: EXEMPT SALES + TAXABLE PURCHASES

> **This is the most critical section for SanskaTots.**

### The Core Problem

You buy inputs with GST:
- Velcro (12%) + Pens (12%) + Packaging (18%) + Printer (18%) = **You pay GST to vendors**

You sell outputs without GST:
- Books = **Exempt from GST** = You collect ₹0 GST from customers

**Can you claim refund of GST paid on purchases?** → **NO** ❌

### Why You Cannot Claim ITC

Under **Section 17(2) of CGST Act**, ITC is blocked if inputs are used to make **exempt supplies**. Books fall under the Exempt category (not zero-rated exports). This means:

```
GST paid on Velcro ₹100 → CANNOT recover this from government
GST paid on Printer ₹18,000 → CANNOT recover this from government
GST paid on Packaging ₹500 → CANNOT recover this from government
```

This GST becomes your **business cost** and is built into your product pricing.

### Nil-Rated vs Exempt vs Zero-Rated — The Critical Difference

| Type | GST Rate | ITC Claimable? | Example |
|------|----------|----------------|---------|
| **Nil-Rated** | 0% | ❌ No | Printed books (4901) |
| **Exempt** | Exempted by notification | ❌ No | Fresh vegetables |
| **Zero-Rated** | 0% | ✅ Yes (refund) | Exports outside India |
| **Non-GST** | Outside GST scope | ❌ No | Alcohol, petroleum |

> **Key Insight:** If you start **exporting** your books internationally (Amazon USA, etc.), those become **Zero-Rated** and you CAN claim ITC refund on all your inputs. This is a significant advantage of international sales.

### What to Do in GSTR-3B

**Section 3.1(c):** Report your total exempt sales value (book sales)

**Section 4(D) — ITC Reversal:**
If you have any mixed use (some taxable + some exempt), you must reverse proportionate ITC using the formula:
```
ITC to Reverse = Total ITC × (Exempt Turnover / Total Turnover)
```

**Section 5:**
Report exempt supplies here:
- 5(1): Exempt inter-state supplies
- 5(2): Exempt intra-state supplies

### What About GST on Amazon Marketplace Fees?

Amazon charges you:
- **Referral fee**: 18% GST applies → You receive B2B invoice from Amazon
- **FBA fees**: 18% GST applies
- **Advertising**: 18% GST applies

Since your output is exempt, **you CANNOT claim ITC on Amazon fees either**. These GST amounts become your cost.

**However:** If you ever sell ANY taxable product alongside books, you can claim proportionate ITC. This is an opportunity (see Section 10).

### Practical Accounting Treatment

```
Purchase Price of Velcro: ₹1,000
GST (12%): ₹120
Total Paid: ₹1,120

In your books:
Dr. Raw Material (Velcro): ₹1,120
Cr. Accounts Payable: ₹1,120

(The full ₹1,120 including GST is your cost — no ITC entry)
```

## 7. FINANCE MAINTENANCE FOR STRESS-FREE GST FILING

### The Golden Rule: Separate Everything

**Bank Accounts:**
- Account 1: Business receipts (Amazon payouts, direct sales)
- Account 2: Business expenses
- Never mix personal and business transactions

### Minimum Records to Maintain

**1. Sales Register (maintain daily)**
```
Date | Invoice No | Customer | State | HSN | Amount | GST | Nature (Exempt/Taxable)
```

**2. Purchase Register (every bill)**
```
Date | Vendor Name | Vendor GSTIN | Invoice No | HSN | Amount | GST Paid | Category
```

**3. Expense Register**
```
Date | Expense Type | Amount | GST Paid | Vendor GSTIN | Can Claim ITC?
```

## 8. COMMON MISTAKES BUSINESS OWNERS MAKE

### Mistake 1: Not Filing NIL Returns ❌
**Problem:** Many think "I had no sales this month, so no need to file."
**Reality:** You MUST file NIL returns every period. Late filing = ₹50/day penalty (₹20 for NIL returns, capped at ₹2,000).

### Mistake 2: Missing the 11th Deadline for GSTR-1
**Problem:** Vendor files GSTR-1 after 11th → Your customer can't see it in GSTR-2B → ITC dispute.
**Reality:** Always file GSTR-1 by 10th to give customers time. This builds trust with B2B clients.

### Mistake 3: Claiming ITC on Personal Expenses
**Problem:** Business owner buys a restaurant meal "for a client meeting" and claims ITC.
**Reality:** Food and beverages ITC is **blocked under Section 17(5)**. Blocked credits include:
- Restaurant/food & beverages
- Club membership
- Personal vehicle (unless in transport business)
- Employee accommodation (residential)
- Beauty treatment
- Health & fitness

### Mistake 4: Not Maintaining E-Way Bills
**Problem:** Moving goods worth >₹50,000 without E-Way Bill.
**Reality:** Goods can be seized + penalty = tax amount or ₹10,000 (whichever is higher).
**For books:** Even though exempt, E-Way Bill is required for movement >₹50,000.

### Mistake 5: Incorrect HSN Code
**Problem:** Filing with wrong HSN code.
**Reality:** Attracts scrutiny, notice, and penalty of ₹50,000. Always verify on GST rate finder.
- HSN code digits required based on turnover:
  - Upto ₹5Cr: 4 digits
  - Above ₹5Cr: 6 digits

### Mistake 6: Not Reconciling GSTR-2B
**Problem:** Vendors file late or with errors → Your ITC in GSTR-2B doesn't match your records.
**Reality:** ITC can only be claimed as per GSTR-2B (post 2021 rules). Follow up with vendors.

### Mistake 7: Cash Sales Without Invoice
**Problem:** Taking cash payment and not issuing invoice to "avoid GST."
**Reality:** Even exempt supplies need to be recorded. Suppression of sales = Fraud under GST = 100% penalty + prosecution above ₹5 Cr.

### Mistake 8: Missing the Composition Scheme Opt-in Window
**Problem:** Eligible businesses miss the April 1 opt-in window.
**Reality:** Opt-in for Composition Scheme must be done by March 31 for next financial year. If missed, pay regular GST for entire year.

### Mistake 9: Not Updating GST Portal with Bank Account / Address Changes
**Problem:** Business changes address/bank but doesn't update GST portal.
**Reality:** Refunds get stuck. Notices go undelivered. Update GST registration within 15 days.

### Mistake 10: Treating Amazon Payout as Total Sales
**Problem:** Amazon pays after deducting fees. Owner records payout as sales.
**Reality:** **Gross sales value** (before Amazon deductions) must be shown in GST returns. Report full MRP × units sold.

## 9. E-COMMERCE SPECIFIC RULES

### TCS (Tax Collected at Source) by E-Commerce Operators
- **Amazon/Flipkart/Meesho** deduct **1% TCS** (0.5% CGST + 0.5% SGST or 1% IGST) from your payout
- This TCS reflects in your **Electronic Cash Ledger** automatically
- You can use this TCS credit to pay your GST liability
- **How to check:** GST Portal → Services → Ledgers → Electronic Cash Ledger

**Important:** Amazon files **GSTR-8** on your behalf with your GSTIN for TCS deducted. You can see this in your GSTR-2A.

### Reporting Sales on Amazon in GSTR-1
- Amazon sales = B2C (usually) since you're selling to end consumers
- Report under **B2C (Others)** section in GSTR-1
- Don't report individual invoices for B2C — aggregate by state is fine for small amounts
- For sales >₹2.5 lakh to a single unregistered buyer → B2C Large (report separately)

### E-Commerce Platforms & GST on Their Services
Amazon charges you fees. They issue GST invoices (18% GST). Example:
```
Referral Fee: ₹100
GST (18%): ₹18
Total Deducted: ₹118
```
Since your output is exempt → **This ₹18 is your cost**, not reclaimable.

### Amazon Brand Registry & Advertising
- Sponsored ads are considered "information technology services"
- 18% GST
- Not claimable for exempt sellers

### What's an E-Commerce GSTIN Requirement?
- Your GSTIN must be registered in **every state you sell to** — NO. This is a myth.
- **Single GSTIN is sufficient** for pan-India e-commerce sales via marketplace (Amazon handles state-wise compliance through their platform registration).

## 10. LEGAL TAX-SAVING STRATEGIES & SMART HACKS

> **Important:** All strategies below are **legal and recognized under GST law**. This is not tax evasion guidance.

### Strategy 1: Export Your Books → Claim Full ITC Refund

**This is the biggest opportunity for your business.**

When you export books to USA, UK, Singapore (Amazon Global / Etsy / your own website):
- Export = **Zero-Rated Supply** under Section 16 of IGST Act
- Even though books are 0%, exports allow you to **claim ITC refund** on all inputs
- All GST paid on Velcro, Pens, Packaging, Printer → **fully refundable**

**How:**
1. File **LUT (Letter of Undertaking)** on GST portal (free, valid for 1 year) before exporting
2. Export without paying IGST
3. Claim refund in GSTR-1 (Table 6A) → Money credited in 60-90 days

**Potential saving:** If you spend ₹5 lakhs/year on taxable inputs at avg 15% GST = **₹75,000 annual refund**

### Strategy 2: Add a Taxable Product Line

Introduce even ONE taxable product (e.g., 12% GST stationery set, activity kit with tools) alongside your books:
- Now you have **mixed supply** (taxable + exempt)
- You can claim **proportionate ITC** on all your inputs
- **Formula:** ITC Claim = Total ITC × (Taxable Turnover / Total Turnover)

If 20% of sales are taxable → claim 20% of all input ITC.

### Strategy 3: Composition Scheme (If Eligible & Applicable)

**Only for manufacturers/traders, not pure service providers.**

| Turnover | Rate |
|----------|------|
| Up to ₹1.5 crore | 1% of turnover |
| Manufacturers | 2% of turnover |
| Restaurants | 5% of turnover |

**Benefits:**
- File only 1 quarterly return (GSTR-4) + 1 annual
- No complex ITC calculations
- Simple flat-rate tax

**Limitations:**
- Cannot do inter-state sales ❌ (kills Amazon all-India sales)
- Cannot issue tax invoices (buyers can't claim ITC from you)
- **Not suitable for your Amazon business** (inter-state required)

### Strategy 4: Claim ITC on Business Infrastructure

If you register a **taxable product** even partially:
- Printer, laptop, office furniture, AC, camera for content creation → All **claimable ITC**
- Mobile phone used for business → Claim 50% ITC (if dual use)
- Internet connection (broadband) → 100% ITC if purely business

### Strategy 5: Business Restructuring — Separate Entities

If your business grows:
- Entity A: SanskaTots (books + exempt goods) — No ITC, simple filing
- Entity B: SanskaTots Design Studio (design services/training) — Taxable, full ITC

Entity B's purchases (printers, computers, raw materials) become fully ITC-claimable.
Services provided to Entity A by Entity B can be priced to shift profits to lower-tax entity.

> **Requires CA guidance at this stage.**

### Strategy 6: RCM (Reverse Charge) on Unregistered Vendors

If you buy from unregistered vendors (small craftsmen, local suppliers):
- Under **Section 9(4)** — RCM applies only for notified categories
- Keep purchase invoices from unregistered vendors properly
- For most goods, RCM is NOT currently mandatory for unregistered B2B under ₹5K/day aggregate — use this for small purchases

### Strategy 7: Input on Capital Goods Timing

When purchasing a printer or expensive equipment:
- Register the equipment purchase invoice in the financial year BEFORE you buy (if still taxable)
- Capital goods ITC can be claimed in full in the year of purchase (no more spreading over 5 years — rule changed)
- If you ever move to a taxable product line, capital goods ITC can be reversed and reclaimed per new rules

### Strategy 8: GST Refund on Inverted Duty Structure

If you ever buy something at 18% GST but sell at 12% or 5% GST:
- **Inverted Duty Refund** under Section 54(3)
- You can file for cash refund of the excess ITC accumulated
- Applicable mostly when you introduce taxable products

### Strategy 9: Place of Business Optimization

If you have operations in multiple states, registering in states with:
- Lower state GST rates (some states offer different thresholds)
- Better e-way bill exemptions for local supplies
This is advanced but worth noting.

### Strategy 10: Maintain a Petty Cash Policy for Unregistered Vendor Purchases

Small purchases from local market (felt, craft materials from unregistered vendors):
- Keep a ₹10,000/month petty cash float
- These purchases have no GST invoice to worry about
- Book as expense at cost price
- Below threshold doesn't attract RCM currently

## 11. GST NOTICES & HOW TO HANDLE THEM

### Common Notice Types

| Notice Type | Reason | Action Required |
|-------------|--------|----------------|
| **GSTR-1 vs GSTR-3B mismatch** | Sales reported in 1 ≠ tax paid in 3B | Reconcile and reply with explanation |
| **ITC mismatch (GSTR-2B vs 3B)** | You claimed ITC not in GSTR-2B | Reverse excess ITC + interest |
| **Scrutiny notice (ASMT-10)** | Discrepancy in return | Reply in 30 days with documents |
| **Demand notice (DRC-01)** | Tax demand raised | Pay or dispute within 3 months |
| **Annual return mismatch** | GSTR-9 vs GSTR-3B mismatch | File amended return or pay diff |

### Response Timeline
- Most notices: Reply within **30 days**
- If you miss: 100% tax demand becomes payable immediately
- Always reply on portal, never ignore

### Key Tip: Check Your GST Portal Every Month
Go to Services → User Services → View Notices and Orders. Many businesses miss notices because they don't log in regularly.

## 12. ANNUAL RETURN (GSTR-9)

### Who Needs to File?
- Compulsory: Turnover > ₹2 crore
- Optional: Turnover ≤ ₹2 crore (but advisable to file anyway)
- **Deadline:** 31st December following end of financial year

### What It Contains
- Consolidated summary of all monthly/quarterly returns filed
- ITC claimed vs ITC available reconciliation
- Outward and inward supplies breakup
- Tax paid summary

### GSTR-9C (Reconciliation Statement)
- Required only if turnover > ₹5 crore
- Must be certified by CA/CMA
- Reconciles audited financial statements with GST returns

### Penalty for Late Filing
- ₹200/day (₹100 CGST + ₹100 SGST)
- Maximum: 0.25% of turnover

6. **File first GSTR-3B** — Report exempt sales, pay ₹0 tax (since books = 0%)
7. Set up **LUT (Letter of Undertaking)** on GST portal to prepare for future exports

### Month 4-6
8. Evaluate adding a **taxable product** (stationery/craft kit) to start claiming partial ITC
9. Consider **exporting** to Amazon USA/UK — converts nil-rated to zero-rated, enables full ITC refund
10. Review pricing to include all input GST costs (since no ITC available)

### Year 1 End
11. File **GSTR-9** annual return
12. Review if Composition Scheme makes sense (unlikely given Amazon inter-state sales)
13. Consult CA for strategic restructuring if turnover crosses ₹50L
